Phase 3

Test AVAX and Deploy

Step 3-1

Get Test AVAX

Fuji Faucet からテスト AVAX を取得

ウォレットアドレスを入力

Request をクリック

build.avax.network/console/primary-network/faucet

Step 3-2

Deploy with Remix

1. Remix IDE を開く

2. PositionTracker.sol をコピー

3. コンパイル (Solidity 0.8.x)

4. Injected Provider - Core を選択

5. ネットワーク: Fuji (C-Chain)

6. デプロイ > アドレスをコピー

Step 3-3

Environment Setup

.env.local を作成

cp .env.local.example .env.local

コントラクトアドレスを設定:

NEXT_PUBLIC_CONTRACT_ADDRESS=0x...
