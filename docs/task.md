# 2Dグリッドゲーム開発タスク

これは、10x10の2Dグリッドゲームを開発するためのタスクリストです。

## フェーズ 1: プロジェクトのセットアップと基本的なグリッドレンダリング
- [x] `app/components/GameBoard` コンポーネントを作成
- [x] CSS Grid を使用して 10x10 のグリッドを表示
- [x] グリッドのスタイルを `GameBoard.module.css` で定義

## フェーズ 2: プレイヤーのレンダリングとキーボード操作
- [ ] `app/components/Player` コンポーネントを作成
- [ ] プレイヤーをグリッド上に表示
- [ ] キーボード（矢印キー）入力をリッスン
- [ ] `useReducer` を使用してプレイヤーのローカル状態（座標）を管理

## フェーズ 3: ウォレット接続
- [ ] 既存の `@avalanche-wallet` を利用してウォレット接続機能を追加
- [ ] 接続ボタンとウォレットアドレス表示を実装
- [ ] 接続状態に応じてUIを更新

## フェーズ 4: スマートコントラクト連携
- [ ] ゲーム開始時にコントラクトからプレイヤーの初期位置を取得
- [ ] プレイヤーが移動するたびにコントラクトの `move` 関数を呼び出す
- [ ] トランザクションのステータス（pending, success, error）をUIに表示

## フェーズ 5: リファインメントとクリーンアップ
